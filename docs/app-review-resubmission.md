## 2) Provide Demo Video Link

Add a demo video link in:
- App Store Connect -> App Review Information -> Notes

Video requirements from App Review:

- Shows all core app features
- Shows permission prompts and flows

## 3) Demo Video Recording Script (Suggested)

1. Show device home screen and open Twist.
2. Show photo picker permission prompt and grant permission.
3. Select a Spotify playlist screenshot from Photos.
4. Enter a playlist name.
5. Tap Convert and show progress screen.
6. Show Apple Music permission prompt and grant permission.
7. Show conversion result:
- created track count
- skipped tracks list (if any)
8. Open Apple Music and show the created playlist.
